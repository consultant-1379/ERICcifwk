Instructions for Adding the SSH Key

run the following commands to add the SSH Key to authorized_keys file for each node.

Section A:

Check if .ssh exists in /root using:

$ ls -lrta

If it already exists, skip to Section B. Otherwise, create the directory with the correct permissions by running the ssh-keygen command to generate a key for that user:

$ ssh-keygen -t rsa

Create authorized_keys file in /root/.ssh/:

$ touch authorized_keys

Make sure your authorized_keys file has the correct permissions:

$ chmod 600 /root/.ssh/authorized_keys

Section B:

Copy and paste the SSH Public Keys below into the remote server's .ssh/authorized_keys using a text editor.
